Buscar um elemento dentro de outro

Achando elementos HTML por Tagname

Neste exemplo buscaremos um elemento dentro de outro pelos métodos getElementsByTagName e por getElementById. Para isso, uma div main foi criada com 2 parágrafos. O script criará uma variável para buscar o conteudo deste. Logo depois outra variável buscara o parágrafo[0] desta div e monstrará uma mensagem na tela
