Exercício para treinar laços FOR e FOR IN

Sua tia, professora, encomendou um trabalho para você. Ela quer que você crie um script que vai pedir o número de alunos da turma (peça via prompt mesmo). Em seguida, vai perguntar qual foi a nota de cada um deles. No final, seu script deve fornecer a média geral da turma. Use array para armazenar as notas, laço FOR para preencher cada elemento do array e um laço FOR IN para calcular a soma das notas (cálculo necessário para calcular a média).
